# 소수 찾기

```
import java.util.Arrays;

class Solution {
  public int solution(int n) {
      int answer = 0;
      
      boolean bools[] = new boolean[n+1]; //1-n까지숫자 = 방번호
      Arrays.fill(bools, false);//아직 삭제안한 숫자들

      
      for(int i =2; i<n+1; i++) {
          if(bools[i] == true) {//이미 삭제된 숫자저장하지 않음
              
              
          }
          else {//0이 아닐 경우
              answer++;
              int j = 1;
              while(j*i < n+1){//배수 모두 지우기
                  bools[i*j] = true;
                  j++;
              }
              
              
          }
      }
      
      return answer;
  }
}
```

# 문자열 내 마음대로 정렬하기

```
import java.util.Arrays;

class Solution {
  public String[] solution(String[] strings, int n) {
      String[] answer = new String[strings.length];
      
      //스트링 배열 돌면서 n번째문자부터 잘라붙이기
      for(int i = 0; i < strings.length; i++) {
          strings[i] = strings[i].substring(n) + strings[i].substring(0, n);
          // System.out.println (strings[i]);
      }
      
      // 잘라서 만든 배열 정렬하기
      Arrays.sort(strings);
      //정렬된 배열 다시 원래대로 잘라붙이기
      for(int i = 0; i < strings.length; i++) {
          int l = strings[i].length();
          answer[i] = strings[i].substring(l-n)+strings[i].substring(0,l-n);
      }
      
      return answer;
  }
}
```
